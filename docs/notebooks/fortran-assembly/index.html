    Fortran and assembly

    Reference: https://gcc.gnu.org/wiki/GFortranHacking

    %%writefile hello.f90
program hello
          print *, "Hello World!"
end program

    Writing hello.f90

    ! gfortran hello.f90

    ! ./a.out

     Hello World!

    ! gfortran -S hello.f90

    ! cat hello.s
